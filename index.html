Tab Communication

  • Client-side applications have local state which can be managed a number of ways, for example localStorage or IndexedDb.
  • One common problem is that these storage options are shared across browser tabs, and it is easy for tabs to become stale.
  • This is why Firebase has only recently introduced multi-tab support
